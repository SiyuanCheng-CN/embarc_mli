MLI API Data
============

This chapter describes the data formats, data layouts and data structures that 
are used in the MLI API. Data formats are the different ways of quantizing and 
storing of data elements into a container. The data layouts describe how these 
elements are stored in a multi-dimensional array in memory. The data structures 
describe how function parameters are grouped into structures.
